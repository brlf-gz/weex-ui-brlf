# wxc-touch-div
## 控件说明
封装了对div的点击添加点击效果

## 属性
| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| color | String | #ffffff | 未点击时的颜色 |
| touchColor | String | #ffff00 | 点击时的颜色 |


## 方法
无

## 事件
* onClick：点击事件


## 用法：
参考[示例](https://github.com/brlf-gz/weex-ui-brlf/blob/master/src/demo/wxc-touch-div/index.vue).
