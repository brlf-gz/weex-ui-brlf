# wxc-switch
## 控件说明
`<wxc-switch>`是个类似 checkbox 的 UI

## 属性
| Prop | Type  | Default | Description |
|:----:|:---:|:-------:|:----------:|
|width|String|100px|宽度|
|height|String|60px|高度|
|checked|Boolean|false|默认选中状态|
|disabled|Boolean|false|是否处于不可交互状态|


## 方法
无


## 事件
* change：改变开关状态时触发该事件。 
    * 事件中 event 对象属性：
      * value: 组件布尔值真或假。
